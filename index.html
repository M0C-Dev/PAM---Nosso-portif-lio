Tecnologias utilizadas

Expo

O Expo é uma plataforma e framework de código aberto para desenvolvimento de aplicativos móveis (Android, iOS) e web com React Native. Ele simplifica o processo, eliminando a necessidade de configurar emuladores ou Android Studio, oferecendo bibliotecas nativas prontas e ferramentas como o Expo Go para visualização imediata.

React

O React Native é um framework de código aberto criado pela Meta (Facebook) para desenvolver aplicativos nativos de Android e iOS usando JavaScript e React. Ele permite criar apps multiplataforma com uma única base de código, oferecendo alto desempenho e interface nativa.
